# Колючка и Ко — интернет-магазин кактусов

Небольшой Next.js (App Router) онлайн-шоп для кактусов, суккулентов, наборов и горшков.

```
 ┌────────────────────────────────────────────┐
 │  Главная — каталог — корзина — чекаут     │
 ├────────────────────────────────────────────┤
 │  🌵  Кактусы, которые проще полюбить,     │
 │      чем поливать.                         │
 │                                            │
 │  [Открыть каталог]  [Как ухаживать]        │
 └────────────────────────────────────────────┘
```

## Что есть

- `/` — витрина: hero, преимущества, хиты подоконника, подписка на письма
- `/catalog` — каталог с фильтром по категориям, поиском и сортировкой
- `/product/[id]` — карточка товара: фото, описание, характеристики, инструкция
- `/cart` — корзина в `localStorage`, расчёт доставки, прогресс до бесплатной
- `/checkout` — оформление: контакты, доставка (курьер / самовывоз / СДЭК), оплата
- `/care` — гайд по уходу + сезонный календарь + развенчание мифов
- `/about` — кто мы и где нас найти

## Запуск

```bash
npm install
npm run dev          # http://localhost:3000

npm run build && npm start
```

## Стек

Next.js 14 (App Router) · React 18 · TypeScript · Tailwind CSS · клиентская корзина в `localStorage`. Без базы данных — товары лежат в `src/lib/products.ts`.
